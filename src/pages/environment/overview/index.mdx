## Cloud Pak for Multicloud Management on Skytap

The DTE Team provides the ability to provision learning environment with the Cloud Pak for Multicloud Management 2.x installed running on a set of virtual machines in Skytap.

After you follow this documentation, you can have a Cloud Pak for Multicloud Management 2.x running in Skytap. During the reservation process, you are able to define the expiration date of your environment, max of 14 days.

Have you tried provisioning your own environment with Cloud Pak for Multicloud Management 2.x on Skytap? If not, here you have all the steps to do it! It is a dedicated/personal environment with no extra cost for you.

If you have any issues or feedback, please share it with tutorial authors. Enjoy it!!
